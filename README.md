# 文件分享平台

![项目状态](https://img.shields.io/badge/status-active-success.svg)

## 简介
这是一个文件分享平台，支持文件上传、文件列表查看和链接分享等功能，方便用户在平台上进行文件的管理和分享。

## 功能特性
- 文件上传：支持用户上传本地文件到平台。
- 文件列表查看：展示已上传的文件列表。
- 分享链接：生成文件分享链接，方便用户分享文件。
